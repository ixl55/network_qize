# 🛡️ CyberExam — Network Security Platform

**CyberExam** is a cutting-edge, interactive educational platform designed to test and enhance knowledge in **Network Security**. Featuring a sleek Cyberpunk-inspired UI, it provides a seamless examination experience with real-time performance tracking.

---

## 🚀 Live Access

Ready to test your skills? Access the live environment here:

[**➔ Launch CyberExam Dashboard**](https://ixl55.github.io/network_qize/)

---

## ✨ Key Features

* **⚡ Adaptive Assessment:** Smooth, logic-based transitions between questions for a focused experience.
* **📊 Instant Analytics:** Real-time calculation of scores and percentage rankings upon completion.
* **📜 Persistence Engine:** Automatically saves your last 5 attempts using `LocalStorage` to monitor your learning curve.
* **🎨 High-End UI:** A futuristic Dark Mode interface built with CSS3 animations and custom properties.
* **📱 Fully Responsive:** Optimized for everything from mobile devices to 4K monitors.

---

## 🛠️ Tech Stack

This project is built with high-performance web technologies:

* **HTML5:** Semantic structure and content.
* **CSS3:** Custom variables, glassmorphism effects, and fluid layouts.
* **JavaScript (ES6+):** Manages exam logic, data processing, and local storage state.

---

## 📂 Exam Syllabus

The current question bank covers critical security domains:
- **IPsec Protocols:** Deep dives into ESP and AH.
- **Defense Mechanisms:** Firewalls, IDS/IPS, and Proxy systems.
- **Wireless Security:** WPA/WPA2/WPA3 and encryption standards.
- **OSI Layer Security:** Protection strategies across the network stack.

---

## ⚙️ Customization

To add your own questions, simply edit the `rawQuestions` array within the `<script>` tag in the `index.html` file:

```javascript
const rawQuestions = [
    {
        q: "Your custom question here?",
        opts: ["Option 1", "Option 2", "Option 3", "Option 4"],
        a: "The correct option"
    }
];
```
